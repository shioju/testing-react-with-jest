Testing React with Jest
=======================

This is the sample code for the 'Testing React with Jest' talk at ReactJS Singapore meetup on 10 Sep 2015.

Installing the dependencies
---------------------------

```
npm install
```

Running the tests
-----------------

```
npm test
```
